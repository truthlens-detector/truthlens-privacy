TruthLens Privacy Policy

Last Updated: May 2026

TruthLens is a browser extension designed to provide real-time media authenticity insights for online video content. This Privacy Policy explains how TruthLens handles information, what data is processed, and how user privacy is protected in accordance with Chrome Web Store Developer Program Policies.

By installing or using TruthLens, you agree to the practices described in this Privacy Policy.

1. Information We Collect

TruthLens is designed with a privacy-first approach.
The Extension does not collect, store, sell, or share:

  • Personally identifiable information
  • Browsing history
  • Passwords or authentication credentials
  • Financial or payment information
  • Personal communications
  • Health-related information
  • Precise location information

TruthLens does not create advertising profiles, track users across websites, or sell user data to third parties.

2. How TruthLens Works

TruthLens analyzes visual characteristics of supported online video content in real time to provide media authenticity insights and synthetic content indicators.

To perform this functionality, the Extension may temporarily access:

  • Video frames
  • Website content associated with the active video page
  • User interaction with the extension interface

This processing occurs only for the purpose of providing the Extension’s core functionality.

TruthLens does not permanently store analyzed video content or maintain records of user browsing activity.

3. Local Processing and Storage

TruthLens primarily operates through temporary real-time analysis.

Limited information may be stored locally on the user’s device using Chrome’s storage functionality, including:

  • Extension preferences
  • User interface settings
  • Temporary operational configuration

This locally stored information remains on the user’s device and is not used for advertising, profiling, or unrelated tracking purposes.

4. Optional Feedback

Users may voluntarily provide feedback to help improve the functionality and performance of TruthLens.

Feedback submissions should not contain sensitive personal information. Any feedback provided is used solely for improving the Extension experience and related analysis systems.

5. Data Sharing and Third Parties

TruthLens does not sell, rent, or transfer user data to third parties.

If server-side processing is used to support analysis functionality, transmitted data is limited strictly to the operational requirements necessary for generating authenticity-related insights.

We do not use collected or processed data for:

  • Advertising purposes
  • User profiling
  • Creditworthiness evaluation
  • Data brokerage
  • Unrelated analytics activities

6. Accuracy and Informational Use

TruthLens provides informational media authenticity insights based on evolving analysis methods.

Results generated by the Extension are probabilistic in nature and may not always be accurate. Users should independently verify important information and should not rely solely on the Extension for factual, legal, financial, or professional decisions.

TruthLens does not guarantee the authenticity, origin, or manipulation status of any content.

7. Security

Reasonable technical and organizational safeguards are used to help protect operational data and Extension functionality.

However, no internet-based system can be guaranteed to be completely secure, and users acknowledge this inherent risk when using online services and browser extensions.

8. Compliance with Chrome Web Store Policies

TruthLens complies with applicable Chrome Web Store Developer Program Policies regarding:

  • Limited use of user data
  • Permission transparency
  • Data minimization
  • User privacy protections

The use of information processed by TruthLens is limited strictly to the functionality described in the Extension’s listing and user interface.

9. Changes to This Privacy Policy

This Privacy Policy may be updated periodically to reflect changes in functionality, legal requirements, or Chrome Web Store policies.

Updated versions will be posted through the official Privacy Policy URL associated with the Extension.

Continued use of TruthLens after updates constitutes acceptance of the revised policy.

10. Limited Use Disclosure

TruthLens adheres to the Chrome Web Store User Data Policy, including the Limited Use requirements.
